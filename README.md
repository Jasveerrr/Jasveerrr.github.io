# Cohort2git

A simple Node.js project created to understand Git and GitHub workflows, including
repository setup, commits, cleanup using `.gitignore`, and best practices.

## Purpose
This project is meant for learning and practicing:
- Git basics (init, add, commit, push)
- Commit history and commit IDs
- Cleaning repositories using `.gitignore`
- Professional repository structure

## Tech Stack
- JavaScript
- Node.js
- Git & GitHub
